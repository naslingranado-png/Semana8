El comando uname -a muestra información del sistema operativo, como el nombre del sistema, la versión del kernel y la arquitectura del equipo.

El comando whoami muestra el nombre del usuario que está utilizando actualmente el sistema.

Conocer esta información es útil para administrar servidores porque permite identificar el entorno de trabajo, verificar permisos de usuario y solucionar problemas técnicos de manera más eficiente.
